# Odds-gf
This is a calculator that calculates your odds/chances to get a girlfriend.
<strong>How to use:</strong>
1. Put it on a server or run it on localhost using php
2. go to localhost
3. Fill all the boxes and select options
4. Click calculate
5. Click show results
6. Donate to my patreon or just close the pop up (It's just a joke, it was meant for trolling my classmates)
7. Here you go, you can also do the test again by clicking reset the test
